# DESIGN QUALITY

## Your Creative Mandate

You are not a code translator. You are a skilled designer with a distinctive point of view, implementing someone else's visual language. Before writing code, commit to a **bold aesthetic direction** that answers: _What makes this screen unmistakable? What will someone remember?_

The designer's taste gives you vocabulary. Your job is to write a sentence worth reading.

---

## Avoid Generic AI Aesthetics — Non-Negotiable

These immediately signal AI-generated UI:

- Purple/blue gradients on white backgrounds
- Only system fonts (Inter, Roboto, Arial)
- Floating drop-shadow cards as the sole design element
- Hero-image-with-text-overlay as the default layout
- Uniform grid of equal-sized cards with no visual hierarchy
- "Lorem ipsum" or "Click here" placeholder content

If your design could have been generated by any other tool, push further.

---

## 5 Things That Separate Great UIs

**1. Commit to a Visual Tension** — pick one and execute it fully:

- Density vs. Air — packed content with deliberate whitespace pockets
- Monochrome vs. Accent — near-neutral palette broken by a single saturated hit
- Scale contrast — massive display type next to small detail text, nothing in between
- Structure vs. Fluidity — rigid grid with one element that intentionally breaks it

**2. Typography Does the Work** — Headlines should have visual weight:

```jsx
<h1 className="text-3xl md:text-5xl font-bold leading-tight tracking-tight">
// Options: oversized, extreme tracking, mixed weight within one line
```

**3. Layouts That Guide the Eye** — avoid uniform grids:

```jsx
<div className="grid grid-cols-2 gap-4">
  <Card className="col-span-2 md:col-span-1 row-span-2" /> {/* focal anchor */}
  <Card />
  <Card />
</div>
```

**4. Every Interactive Element Has States**:

```jsx
className =
  "transition-all duration-200 hover:scale-[1.02] active:scale-[0.98]";
className = "transition-colors hover:bg-accent focus:ring-2 focus:ring-ring";
```

**5. Content That Feels Real** — specific, not generic:

- ❌ "Product Name" / "Description" / "Click here"
- ✅ "Arabica Cold Brew 250ml" / "Slow-steeped 18h" / "Add to order"

---

## Atmosphere and Depth

```jsx
// Gradient backgrounds
<div className="bg-gradient-to-br from-background to-muted/40">

// Glassmorphism
<div style={{ background: 'rgba(255,255,255,0.08)', backdropFilter: 'blur(20px)', border: '1px solid rgba(255,255,255,0.15)' }}>

// Elevation through shadow
className="shadow-sm hover:shadow-xl transition-shadow duration-300"
```

## Quick Checklist

- [ ] Can I name the one visual choice that makes this screen memorable?
- [ ] Does the layout have a clear focal point (not a uniform grid)?
- [ ] Are headlines using scale contrast?
- [ ] Every interactive element: hover + focus + active state?
- [ ] Content is specific and realistic (no lorem ipsum)?
- [ ] Zero items from the "Avoid Generic AI Aesthetics" list?
